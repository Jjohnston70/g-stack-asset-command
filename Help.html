Dashboard Guide

Opening the Dashboard

Go to menu: 🚗 Tools > Open Dashboard

The dashboard opens in a popup window with interactive charts and real-time data.

Dashboard Views

The dashboard has four main views, accessible via the colored buttons at the top:

Assets View (Blue)

  • Total Assets - Count and total value of all assets
  • Available/In Use - Current availability status
  • In Maintenance - Assets currently being serviced
  • YTD Costs - Year-to-date maintenance and fuel costs
  • Charts - Asset status distribution and types breakdown
  • Activity Feed - Recent check-outs, check-ins, and refuels

Maintenance View (Yellow/Orange)

  • Overdue - Count of overdue maintenance items
  • Due This Week - Upcoming maintenance within 7 days
  • Scheduled - Total pending maintenance
  • Completed (YTD) - Maintenance completed this year
  • Lists - Detailed overdue and upcoming items

Compliance View (Orange/Red)

  • Active Drivers - Number of active drivers
  • Expired - Credentials that have expired
  • Expiring Soon - Credentials expiring within 30 days
  • Clearinghouse Issues - Drivers with DOT clearinghouse flags
  • Driver Table - Status of all driver credentials

Fuel View (Purple)

  • Total Fuel (30 Days) - Gallons consumed
  • Total Cost - Fuel expenses
  • Avg Cost/Gallon - Average fuel price
  • Anomalies - Unusual fuel consumption patterns
  • Charts - Top 5 consumers by fuel and cost

Sending Alert Emails

Each view has a "Send Alert Email" button at the bottom. Clicking it sends an email to the configured alert address with relevant information for that view.

Refresh Data

Click the blue "Refresh" button to reload all dashboard data with the latest information from your sheets.

Manual Data Entry

Opening Data Entry

Go to menu: 🚗 Tools > Data Entry

A sidebar opens on the right with four tabs: Assets, Drivers, Maintenance, and Costs.

Adding Assets

Field Required Description
Asset ID Yes Unique identifier (e.g., A-001, VEH-101)
Asset Name Yes Descriptive name (e.g., "Red F-150", "Forklift #3")
Type Yes Vehicle, Equipment, Tool, Trailer, or Other
Status Yes Available, In Use, Maintenance, or Retired
Current Location No Where the asset is currently located
Assigned To No Employee currently using the asset
Last Service Date No Date of most recent service
Service Interval No Days between required services
Acquisition Date No When the asset was purchased
Acquisition Cost No Purchase price in dollars

Adding Drivers

Field Required Description
Driver ID Yes Unique identifier (e.g., DRV-001)
Driver Name Yes Full name of the driver
Status Yes Active, On Leave, Suspended, or Terminated
Phone No Contact phone number
Email No Contact email address
License Number No State driver's license number
License Expiry No When the license expires (tracked for alerts)
CDL Number No Commercial Driver License number
CDL Expiry No When CDL expires (tracked for alerts)
Medical Card Expiry No DOT medical certificate expiration

Adding Maintenance Records

Field Required Description
Maintenance ID Yes Unique identifier (e.g., M-001)
Asset ID Yes ID of the asset being serviced
Service Type Yes Type of service performed
Scheduled Date Yes When service is/was scheduled
Completed Date No When service was completed (leave blank if pending)
Parts Cost No Cost of parts in dollars
Labor Cost No Cost of labor in dollars
Vendor No Service provider name
Invoice Number No Vendor invoice reference

Adding Activity/Cost Records

Field Required Description
Asset ID Yes ID of the asset
Action Type Yes Check-out, Check-in, Refuel, Maintenance, Incident, Location Update
Employee No Who performed the action
Location No Where the action occurred
Odometer/Hours No Current meter reading
Fuel Gallons No* Amount of fuel added (for Refuel only)
Fuel Cost No* Total fuel cost (for Refuel only)
Auto-Status Updates

When you log a Check-out, the asset status automatically changes to "In Use". Check-in changes it back to "Available".

CSV Tips

Date Format

Always use YYYY-MM-DD format (e.g., 2024-12-25). This ensures consistent date parsing.

Numbers

Don't include currency symbols ($) or thousand separators (,) in number fields. Use plain numbers like 35000 not $35,000.

Text with Commas

If your text contains commas, wrap it in double quotes: "Smith, John"

Excel Users

When saving from Excel, choose "CSV (Comma delimited) (*.csv)" format. Excel may change date formats - double-check them before importing.

Sheet Reference

Assets Master

Central repository of all tracked assets.

  • Each asset needs a unique Asset ID
  • Status determines availability tracking
  • Service dates trigger maintenance alerts
  • Acquisition data used for cost tracking

Drivers

DOT compliance tracking for all drivers.

  • Tracks license, CDL, and medical card expirations
  • Monitors clearinghouse status
  • HOS (Hours of Service) and ELD compliance fields
  • DVIR (Driver Vehicle Inspection Report) tracking

Activity Log

Records all asset usage events.

  • Check-out/Check-in logs
  • Fuel purchase records
  • Maintenance events
  • Location updates and incidents

Maintenance Tracker

Scheduled and completed maintenance records.

  • Schedule upcoming maintenance
  • Track parts and labor costs separately
  • Store vendor and invoice information
  • Status automatically tracked (Scheduled/Completed/Overdue)

Cost Tracking

Aggregated cost data per asset (auto-calculated).

  • 30-day fuel costs
  • YTD maintenance costs
  • Trip counts and averages

Config

System settings and preferences.

Fuel Anomaly %Threshold for flagging unusual fuel consumption (default: 20%)

Troubleshooting

Common Issues

"Sheet not found" Error

Cause: Required sheets haven't been created.

Solution: Run 🚗 Tools > 1. Build Complete Template first.

Dashboard Not Loading

Cause: Pop-up blocker or script authorization.

Solution:

  1. Allow pop-ups for Google Sheets
  2. If prompted, authorize the script to run
  3. Try refreshing the page and reopening

CSV Import Failed

Cause: Incorrect CSV format.

Solution:

  • Download the template and match the format exactly
  • Check date format is YYYY-MM-DD
  • Remove any $ symbols from numbers
  • Ensure no extra blank columns

Emails Not Sending

Cause: Email not configured or daily quota exceeded.

Solution:

  • Check Config sheet has valid email address
  • Google limits emails to ~100/day for free accounts
  • Check spam folder

Maintenance Status Not Updating

Cause: Status column needs manual update or formula refresh.

Solution:

  • For scheduled items, add a Completed Date to mark as complete
  • Use Refresh Dashboards from menu to update calculations

Asset Status Not Changing on Check-in/out

Cause: Asset ID doesn't match exactly.

Solution:

  • Ensure Asset ID in Activity Log matches Assets Master exactly
  • IDs are case-sensitive (A-001 is different from a-001)
